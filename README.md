Milkshakebar PHP Project

This is a PHP and MySQL based Milkshakebar website developed using XAMPP.

Features:
- User registration and login
- Milkshake product listing
- Add to cart
- Order placement
- Admin order management

Technologies Used:
- PHP
- MySQL
- HTML
- CSS
- XAMPP

How to Run:
1. Install XAMPP
2. Copy project folder into htdocs
3. Import milkshakebar.sql into phpMyAdmin
4. Run in browser:
   http://localhost/myproject
